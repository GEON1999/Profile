## 1. nextjs 13 layout

```jsx
/* ./app/layout */

<html>
      <nav>
        <div>Geon</div>
        <ul className="w-full flex flex-row">
          <li className="text-red-600">ABOUT</li>
          <li>STACK</li>
          <li>WORK</li>
          <li>CONTACT</li>
        </ul>
      </nav>
      <body>
        <div>{children}</div>
      </body>
    </html>
```

- 최초에 위와 같이 `nav` 를 코딩 했다.
- 위와 같이 코딩시 `children` prop 이 `nav` 위에 위치를 했다.

```jsx
/* ./app/layout */

<html>
      <body>
        <nav>
          <div>Geon</div>
          <ul className="w-full flex flex-row">
            <li className="text-red-600">ABOUT</li>
            <li>STACK</li>
            <li>WORK</li>
            <li>CONTACT</li>
          </ul>
        </nav>
        <div>{children}</div>
      </body>
    </html>
```

- 위와 같이 `body` 내에 layout `element` 와 `children` 을 순서대로 코딩함으로서 해결했다.
